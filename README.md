# CAN-FD Transceiver MCP2562FD Breakout Board #

This board lays out the Microchip Technology MCP2562FD CAN-FD Transceiver IC onto a board to create an external breakout board style CAN physical layer for MCUs with an embedded CAN controller but no transceiever.  

Exact Reference Manufacturer Product Number: MCP2562FD-E/SN
